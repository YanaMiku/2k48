# 🎮 Merge Puzzle Game - Ultimate Edition

Sebuah permainan puzzle merging angka interaktif yang dapat dimainkan di berbagai perangkat. Game ini terinspirasi dari gameplay 2048 dengan sentuhan modern dan fitur yang lebih lengkap.

## ✨ Fitur Utama

- **Grid Variabel** - Pilih ukuran grid dari 4x4 hingga 8x8
- **Kontrol Multi-Platform** - Support touch (mobile) dan mouse (desktop)
- **Sistem Undo** - Kembalikan langkah sebelumnya
- **Pelacakan Skor** - Hitung skor dan jumlah gerakan
- **Animasi Halus** - Efek spawn dan merge tile yang memuaskan
- **Responsif** - Tampilan optimal di semua ukuran layar
- **Icon Library** - Menggunakan Font Awesome 6 (tanpa emoji)

## 🎯 Cara Bermain

1. **Geser/Drag** ubin ke arah yang diinginkan (atas, bawah, kiri, kanan)
2. **Gabungkan** dua ubin dengan angka yang sama untuk mendapatkan angka dua kali lipat
3. **Dapatkan skor** tertinggi dengan menggabungkan ubin sebanyak mungkin
4. **Game berakhir** ketika grid penuh dan tidak ada lagi kemungkinan penggabungan

## 🕹️ Kontrol

| Kontrol | Fungsi |
|---------|--------|
| Geser layar (mobile) / Drag mouse (desktop) | Memindahkan ubin |
| Tombol panah keyboard | Memindahkan ubin |
| Tombol "Game Baru" | Memulai permainan baru |
| Tombol "Undo" | Membatalkan langkah terakhir |
| Dropdown Grid | Mengubah ukuran grid (4x4 hingga 8x8) |

## 🎨 Tampilan Visual

Setiap nilai ubin memiliki warna gradien yang berbeda:
- **1, 2, 4** - Nuansa orange/kuning (nilai rendah)
- **8, 16, 32, 64** - Warna biru, hijau, pink (nilai menengah)
- **128, 256, 512, 1024, 2048+** - Warna cerah (nilai tinggi)

## 📱 Responsivitas

Game ini dirancang untuk bekerja optimal di:
- **Desktop** - Layar lebar dengan kontrol mouse dan keyboard
- **Tablet** - Layout menyesuaikan dengan ukuran layar
- **Mobile** - Kontrol sentuh yang responsif dan ukuran ubin yang proporsional

## 🛠️ Teknologi yang Digunakan

- HTML5
- CSS3 (dengan Flexbox, Grid, Animations)
- JavaScript (ES6+)
- Font Awesome 6 (Icon Library)

## 🚀 Cara Menjalankan

1. Simpan file HTML dengan ekstensi `.html`
2. Buka file tersebut menggunakan web browser modern (Chrome, Firefox, Safari, Edge)
3. Pastikan koneksi internet aktif untuk memuat Font Awesome (atau gunakan versi lokal)

Atau bisa juga dijalankan melalui:
- Local server (XAMPP, Live Server VSCode)
- Hosting web (GitHub Pages, Netlify, Vercel)

## 💡 Tips Bermain

- Fokus pada menggabungkan ubin di sudut untuk mengontrol grid
- Gunakan tombol Undo jika langkah kurang optimal
- Grid yang lebih kecil (4x4) lebih mudah tetapi cepat selesai
- Grid yang lebih besar (8x8) memberikan tantangan lebih lama
- Perhatikan posisi ubin bernilai tinggi agar tidak terisolasi

## 🎯 Skor

Skor dihitung berdasarkan nilai ubin yang digabungkan. Contoh:
- Menggabungkan 1 + 1 = 2 (mendapat 2 poin)
- Menggabungkan 2 + 2 = 4 (mendapat 4 poin)
- Menggabungkan 4 + 4 = 8 (mendapat 8 poin)
- Dan seterusnya...
